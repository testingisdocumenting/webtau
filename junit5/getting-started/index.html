Integration Benefits

WebTau can be used as is, without any special integration with https://junit.org/junit5/ JUnit5.However, integration provides: junit5/reporting Rich Reporting Additional junit5/dynamic-tests Dynamic Tests Capabilities

Maven Import

<dependency> <groupId>org.testingisdocumenting.webtau</groupId> <artifactId>webtau-junit5</artifactId> <version>2.2</version> </dependency>
