Fake Server Creation

Use to create a server with controlled responses def router = server.router() // router-example .get("/hello/:name") { request -> server.response([message: "hello ${request.param("name")}"]) } // router-example .get("/bye/:name") { request -> server.response([message: "bye ${request.param("name")}"]) } // router-example // router-example def myServer = server.fake("my-rest-server", router) // router-example http.get("${myServer.baseUrl}/hello/person") { message.should == "hello person" } http.get("${myServer.baseUrl}/bye/person") { message.should == "bye person" }

Server Slowdown

Use to mark server as unresponsive myServer.markUnresponsive() code { http.get("${myServer.baseUrl}/hello/person") { message.should == "hello person" } } should throwException(~/Read timed out/)

Server Break

Use to mark server as broken myServer.markBroken() http.get("${myServer.baseUrl}/hello/person") { statusCode.should == 500 body.should == null }

Server Fix

Use to remove broken and/or slowdown state myServer.fix() http.get("${myServer.baseUrl}/hello/person") { message.should == "hello person" }
