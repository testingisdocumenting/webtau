Serve Static Content

Use to start a server on random port that will host static content from the specified directory. def myServer = server.serve("my-server", "data/staticcontent") { "id": "user-one", "type": "person" } http.get("${myServer.baseUrl}/data.json") { body.type == "person" }

Host Html For Browser

Use static server to host html files and then open them using a browser. <body> <p>hello</p> </body> browser.open("${myServer.baseUrl}/hello.html") $("p").should == "hello"

Override Response

Use to modify response of a proxied server def router = server.router() .get("/hello/:name") {request -> server.response([message: "hello ${request.param("name")}"]) } myServer.addOverride(router)

Server Slowdown

Use to mark server as unresponsive staticServer.markUnresponsive() code { http.get("${staticServer.baseUrl}/hello.html") } should throwException(~/Read timed out/)

Server Break

Use to mark server as broken staticServer.markBroken() http.get("${staticServer.baseUrl}/hello.html") { statusCode.should == 500 body.should == null }

Server Fix

Use to remove broken and/or slowdown state staticServer.fix()
