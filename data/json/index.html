Read List

Use data.json.list to read data as java.util.List from JSON file. Passed path is either relative based on working dir or absolute file path. Or it can be a resource class path. def list = data.json.list("data/flat-list.json") list[0].name.should == "hello" list[1].payload.info.should == ~/id2 payload/

Read Map

Use data.json.map to read data as java.util.Map from JSON file. Passed path is either relative based on working dir or absolute path. Or it can be a resource class path. def map = data.json.map("data/root-map.json") map.payload.info.should == "additional id1 payload"
